# Grocery Discount Calculator 🛒🐍

A simple Python-based tool designed to automate the checkout process by calculating total costs and applying discounts based on purchase thresholds.

## 🚀 Features
- **Dynamic User Input:** Captures customer name, item details, cost, and quantity.
- **Automated Discount Logic:** - 10% discount for purchases above $100.
  - 6% discount for purchases above $60.
- **Formatted Output:** Displays a clean summary of the transaction with calculated totals.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Concepts:** Conditional Logic (`if-elif-else`), F-strings, User Input Handling.

## 📂 How to Run
1. Ensure you have Python installed.
2. Clone this repository.
3. Run the script using: `python grocery_discount_calculator.py`
